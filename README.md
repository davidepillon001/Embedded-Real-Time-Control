# ERTC — Embedded Real-Time Control Labs

STM32F767ZI (Nucleo-144) lab projects developed during the Embedded Real-Time Control course at the University of Padova.

## Labs

| Folder | Topic |
|--------|-------|
| `lab0/` | STM32CubeMX configuration intro, TurtleBot GPIO setup |
| `lab1/` | UART communication and I2C peripheral control |
| `LAB1_Base/` | STM32CubeIDE base project for Lab 1 |
| `lab2/` | IMU (inertial measurement unit) + servo motor control via SPI/I2C |
| `LAB3_MotorControl_Base/` | DC motor speed and position control (PWM, encoder feedback) |

## Hardware

- **MCU:** STM32F767ZI (ARM Cortex-M7, 216 MHz)
- **Board:** ST Nucleo-144
- **Toolchain:** STM32CubeIDE, STM32CubeMX, HAL drivers
